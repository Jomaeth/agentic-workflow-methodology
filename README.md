# Agentic Workflow Methodology

This repository contains a reusable methodology skill for turning rough business workflow ideas into management-ready, PRD-ready AI workflow definitions.

The core idea:

```text
Rough workflow idea
-> Trigger / cycle
-> Data owners
-> Input sources
-> AI tasks
-> Human review gates
-> Approval
-> Distribution
-> Archive
-> Business value
-> PRD-ready workflow definition
```

## What Is Included

- `project-skills/pipeline-infographic-builder/SKILL.md`
- `project-skills/pipeline-infographic-builder/agents/openai.yaml`
- Four infographic benchmark examples under `assets/reference-benchmarks/`

## What Is Not Included

This public repository intentionally excludes private course materials, source references, Excel workbooks, client documents, tender/commercial data, and internal project files.

## Skill Purpose

The `pipeline-infographic-builder` skill helps management and domain experts define the workflow before a PRD is written. It is designed to clarify:

- Workflow trigger and timing
- Data ownership
- AI task boundaries
- Human review gates
- Approval and sign-off logic
- Distribution and archive process
- Business value and success metrics

## Methodology Position

This skill is the discovery layer before technical development:

```text
Management idea / domain pain point
-> Pipeline infographic
-> Domain expert review
-> PRD
-> Technical planning
-> Build / demo
```

The skill does not make AI the final decision-maker. It preserves the governance principle:

```text
AI Drafts -> Human Checks -> Human Signs
```
